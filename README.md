# Student Performance Analysis

##  Project Overview

This project analyzes student performance data to find patterns and insights.

##  Tools Used

* Python 
* Pandas
* Matplotlib / Seaborn

##  Features

* Data cleaning
* Data visualization
* Performance comparison
* Insights generation

##  Files

* `student_analysis.ipynb` → Main notebook

##  How to Run

1. Open the notebook in Jupyter
2. Run all cells step by step

To understand how different factors affect student performance.
